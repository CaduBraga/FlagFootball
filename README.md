# Flag Football - Página Web

Este projeto é uma página web simples dedicada ao **Flag Football**, um esporte de flag football inspirado no futebol americano. A página oferece informações sobre o esporte, serviços, cadastro de times e um formulário de contato.

## Funcionalidades

- **Sobre o Flag Football**: Explicação sobre o esporte, suas regras e a diferença em relação ao futebol americano tradicional.
- **Serviços**: Descrição dos serviços oferecidos, como treinamentos, torneios e venda de equipamentos.
- **Cadastro de Times**: Formulário para cadastro de times interessados em participar de eventos ou treinos.
- **Contato**: Formulário para o usuário enviar mensagens para a equipe, com campos de nome, e-mail e mensagem.

## Layout

O layout do projeto possui duas versões:
1. **Versão Padrão**: Uma estrutura simples e direta.
2. **Versão Alternativa**: Um layout moderno com **cartões**, **efeitos de hover**, e **design responsivo**. A versão alternativa foi desenvolvida utilizando uma grid flexível para tornar a página acessível em dispositivos móveis e desktops.

## Tecnologias Utilizadas

- **HTML**: Estruturação da página.
- **CSS**: Estilização e design responsivo.
- **Responsividade**: Layout que se adapta a diferentes tamanhos de tela (mobile-first).
